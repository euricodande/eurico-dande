# Personal Portfolio Website

A modern personal portfolio website built with **Next.js**, **TypeScript**, and **Tailwind CSS**.
The project showcases my skills, selected projects, contact information, and background as a Computer Science student specialising in Software Engineering.

## Overview

This portfolio was created to present my work, technical skills, and learning journey clearly and professionally. It is designed to be simple, responsive, and easy to navigate for recruiters, lecturers, internship coordinators, and potential collaborators.

## Features

* Responsive layout for desktop, tablet, and mobile screens
* Light and dark mode support
* Smooth scrolling navigation
* Project showcase section
* Skills and technologies section
* Contact section
* Clean and minimal user interface
* Built with reusable React components

## Tech Stack

* **Next.js**
* **React**
* **TypeScript**
* **Tailwind CSS**
* **CSS Variables**
* **Local Storage** for theme preference

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/euricodande/eurico-dande.git
```

### 2. Navigate into the project folder

```bash
cd eurico-dande
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

Open the project in your browser:

```bash
http://localhost:3000
```

## Customization

You can customise the portfolio by editing the component files inside the `components` folder.

Common areas to update:

* Personal introduction in `Header.tsx`
* About section in `About.tsx`
* Skills and services in `Services.tsx`
* Projects in `Work.tsx`
* Contact details in `Contact.tsx`
* Theme colours and fonts in `globals.css`

## Purpose of the Project

The main purpose of this project is to create a professional online presence while applying modern frontend development practices. It also serves as a practical project for improving my understanding of component-based development, responsive design, TypeScript, Tailwind CSS, and Next.js.
